Welcome to the Student Management System

This system allows you to manage student information efficiently. You can perform the following actions:

  • Create new student records with details like name, age, and course.
  • Read and view the list of all students in the system.
  • Update student information, including name, age, and course.
  • Delete student records if needed.
  • Apply filters and search to quickly find specific students based on criteria.

Get started by navigating to the Student Management section using the menu above.
